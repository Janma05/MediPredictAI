# MediPredict AI

Healthcare Management System built using Flask, SQLite and Machine Learning.

## Features
- Add Patient
- View Patient
- Edit Patient
- Delete Patient
- Search Patient
- Export CSV
- Risk Prediction
- Dashboard Analytics

## Technologies
- Python
- Flask
- SQLite
- Scikit-Learn
- HTML
- CSS
- Bootstrap

## Run Project

pip install -r requirements.txt

python app.py
