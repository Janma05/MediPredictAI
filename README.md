# MediPredict AI

Healthcare Management System with Machine Learning-based Health Risk Prediction.

## Project Overview

MediPredict AI is a web-based healthcare management system that allows users to manage patient records and predict patient health risk levels using machine learning.

## Features

* Add New Patient
* View Patient Details
* Edit Patient Records
* Delete Patient Records
* Search Patients by Name or Email
* Export Patient Data to CSV
* Health Risk Prediction
* Dashboard Analytics
* SQLite Database Integration

## Technologies Used

* Python
* Flask
* SQLite
* Scikit-Learn
* Pandas
* NumPy
* HTML5
* CSS3
* Bootstrap 5

## Installation

Clone the repository:

git clone https://github.com/Janma05/MediPredictAI.git

Navigate to project folder:

cd MediPredictAI

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py

Open browser:

http://127.0.0.1:5000
